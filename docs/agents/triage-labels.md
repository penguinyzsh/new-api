# 分诊标签

相关技能使用五种标准分诊角色。下表将这些角色映射到本仓库 Issue 跟踪器中的实际标签。

| 标准角色 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可由无人值守 Agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

当技能提到某种角色，例如“应用可由 Agent 处理的分诊标签”时，必须使用上表中对应的实际标签。
